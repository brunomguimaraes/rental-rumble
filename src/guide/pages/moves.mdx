# Move pools

Forget the four-move limit: every Pokémon walks in with up to **{MOVE_SLOTS}
moves** and the AI decides which to throw each turn.

A pool layers:

- Its own **STAB attacks** — moves that share the user's type. Each move is
  **Physical** or **Energy**, so a mon's stronger attack stat decides which hits
  land hardest.
- **Element-themed coverage**, which shifts with the sign, so the same species
  plays differently run to run.
- A **priority jab** for fast attackers.
- Exactly **one setup button**, matched to the mon: **Swords Dance** / **Nasty
  Plot** for physical / energy attackers, **Dragon Dance**, **Bulk Up** or
  **Calm Mind** for the right typings.
- Exactly **one defensive button** for bulky mons — **Iron Defense** or
  **Amnesia** (fortifying the guard it's already better at) or **Recover** (for
  HP-heavy ones), never two at once.

Hard-hitting attackers instead pack anti-wall tools — **Super Fang**
(DEF-ignoring chip) and **Taunt** (stall-breaker).
